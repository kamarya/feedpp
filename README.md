# Feed Plus Plus
This is a C++ feed parser library. It is an extract of
[Newsbeuter](https://github.com/akrennmair/newsbeuter)
project that contains only the feed parser.
## Build and Install
```
mkdir release
cd release
../configure
make
make install
```
